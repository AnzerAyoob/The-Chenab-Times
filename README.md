## About

This repository is an editorial archive for The Chenab Times — an independent English and Urdu news outlet covering politics, identity, culture, and environment across India.

---

## Folder Structure

| Folder | Contents |
|--------|----------|
| `featured/` | Featured articles, auto-synced from RSS |
| `op-ed/` | Opinion and editorial pieces, auto-synced from RSS |
| `drafts/` | Work in progress (added manually) |
| `scripts/` | Automation scripts |

---

## Auto-Sync

Articles are automatically fetched from the live RSS feeds every 6 hours via GitHub Actions and saved as Markdown files organized by year (e.g. `featured/2026/`).

To trigger a manual sync:
**Actions → Sync RSS Feed to Repo → Run workflow**
